# セットアップ手順

## xcodeの設定
1. xcodeをappstoreからDW

2. Command Line Toolsの導入

```
# ターミナル.appで下記を実行
$ xcode-select --install
```

## homebrewの設定
1. homebrew.shをローカル環境に作成し実行

2. ローカル環境でdotfilesディレクトリをclone

```
$ mkdir dotfile
$ cd dotfile
$ git clone ~~~~~
```

## preztoの設定
1. setup.shを実行

## neovimの設定
1. neovim.shを実行

## nodebrewの設定
1. nodebrew.shを実行
